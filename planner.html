How this works

  1. Drop an activity suggestion in the Activity Suggestions thread.
  2. Reply to discuss details, costs, and logistics.
  3. Vote by reacting with 👍 to comments you support.
  4. Use the Timeline Poll thread to vote on when each activity should happen between June 24 and June 27.

You will need a GitHub account to comment and vote.

1) Activity Suggestions + Discussion + Voting

Post one activity per comment. Add links, expected cost, and best time of day. Everyone can reply and react with 👍 to vote.

Need direct replies? Open the full GitHub thread and use Quote reply or Reply under the specific comment.
